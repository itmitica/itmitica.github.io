Blast off!

I decided it's time I got up close and personal and a bit vocal about the tech I like most.

The backend

Embracing the start simple philosophy, I decided upon serving static pages. I like nodejs for this on my workstation, and harp, the static webserver with built-in jade and sass preprocessing.

The frontend

Following the same philosophy, I start with initilizr. A few tweaks, my own paletton aided blue theme, my own inkscape based graphics, such as my logo, google fonts for richer typography and that's pretty much where it all begins.

Disqus for active feedback and it seems like I've got this covered.

My content

Content is king.
And raising a king requires a lot of work. Wish me luck :)
